What is TagSpeak?

TagSpeak is a symbolic, dataflow-oriented DSL where everything is a packet of the form [op@arg]. Packets chain left â†’ right with >, values flow through, and blocks group work with { ... }.

  • Everything is a packet â†’ [op@arg]
  • Packets chain with > â†’ [math@2+2] > [store@x] > [print@x]
  • Blocks use { ... } to group sequences
  • Strings use quotes â†’ [print@"hello"]
  • Comments welcome: #, //, /* ... */, or TagSpeakâ€™s own [note@"..."]
